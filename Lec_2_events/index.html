Event binding with JS

addEventListner

object.addEventListener(event, handler, capturePhase);
  • Object is the node to which the listener is to be bound.
  • Event is a string indicating the type of event.
  • Handler is the function that should be called when the event occurs.
  • capturePhase is a Boolean indicating whether to use Bubbling (false) or Capture (true). This is optional. If you omit there is false by default.
Ex:- btn.addEventListner("click", show, false);

Why to use addEventListener()

It allows you to bind multiple handlers to an object for the same event. It can work on any DOM Element, not just HTML one

Phases

  • Capture Phase(Top to Bottom) (use true in third parameter of addEventListener() fun)
  • Target Phase(That element which you target)
  • Bubbling phase(Bottom to top) (use false in third parameter of addEventListener() fun)
